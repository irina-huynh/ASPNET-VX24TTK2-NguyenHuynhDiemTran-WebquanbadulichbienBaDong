# WEBSITE GIỚI THIỆU DU LỊCH BIỂN BA ĐỘNG

## Giới thiệu

Website Giới thiệu Du lịch Biển Ba Động là đồ án được xây dựng bằng **ASP.NET MVC 5** kết hợp **SQL Server** nhằm quảng bá hình ảnh du lịch Biển Ba Động, tỉnh Trà Vinh.

Hệ thống cung cấp thông tin về các điểm tham quan, lịch trình tham quan, công ty du lịch liên kết, đánh giá của khách hàng và các thông tin hỗ trợ khách du lịch. Ngoài ra hệ thống còn có trang quản trị giúp quản lý dữ liệu một cách thuận tiện.

## Hướng dẫn cài đặt

1. Clone repository.
2. Mở project bằng Visual Studio 2022/2026.
3. Khôi phục Database bằng file:

```
setup/BaDongTour.sql
```

4. Cập nhật chuỗi kết nối trong Web.config.
5. Chạy website bằng IIS Express.
